# HelloApp

## Overview
HelloApp is a simple application created to demonstrate basic Git and GitHub workflows including branching, pull requests, and project organization.

## Purpose
The project implements different use cases such as:
- Displaying Hello World
- Displaying the user name
- Other greeting features

## Project Structure

src/ - Source code  
bin/ - Compiled binaries  
docs/ - Documentation files  

## How to Run

1. Navigate to the src folder
2. Compile the program
3. Run the compiled file

Example:

javac Hello.java  
java Hello
